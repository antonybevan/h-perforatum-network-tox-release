\section*{Data Availability}

All processed data supporting this study (the committed liver-interactome networks, the curated target and DILI gene sets, and the computed result tables, with SHA-256 checksums in \path{data/CHECKSUMS.sha256}) are openly available in the project repository and in an archived DOI-linked snapshot (see Code Availability). \textit{The repository URL and DOI are withheld in this anonymized version because they identify the authors, and are provided in the identified submission files for editorial verification.}

Computed source data underlying the figures and tables are provided in the anonymized supplementary repository; key tabulated summaries are included in the Supplementary Information. Raw data were obtained from the following public repositories:
\begin{itemize}
    \item STRING v12.0: \url{https://string-db.org}
    \item GTEx v8: \url{https://gtexportal.org}
    \item ChEMBL v31: \url{https://www.ebi.ac.uk/chembl}
    \item DisGeNET: \url{https://www.disgenet.org}
    \item DILIrank 2.0: \url{https://www.fda.gov/science-research/ltkb}
\end{itemize}
