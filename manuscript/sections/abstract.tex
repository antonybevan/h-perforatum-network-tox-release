\section*{Abstract}

Comparing network-proximity Z-scores across compounds with unequal target counts can reverse the evidence ranking relative to the raw topological effect: the larger set's null standard deviation shrinks as $|T|^{-1/2}$, a consequence of the law of large numbers on an averaged statistic, so weaker proximity can register as stronger standardized evidence. We demonstrate this dissociation using two \textit{Hypericum perforatum} constituents in a liver-expressed interactome. Hyperforin (10 targets) is closer to the drug-induced liver injury (DILI) module than Quercetin (62 targets) ($d_c = 1.30$ vs $1.68$, STRING $\geq$900) yet receives the weaker proximity Z-score ($-3.86$ vs $-5.44$); the evidence ranking reverses between network thresholds (Hyperforin $Z = -6.04$ vs Quercetin $-5.46$ at STRING $\geq$700), while the raw-distance ranking does not. We trace this to the $|T|^{-1/2}$ null-precision law, which a degree-controlled calibration benchmark confirms holds across the DILI module and size-matched pseudo-modules and shows that material rank reversal is a conditional outcome requiring the smaller set to be above the 90th percentile of probe-pair margins, while unconditionally Z-score and raw-distance rankings disagree in $\sim$12\% of cross-size comparisons at $R=6.2$. As a complementary effect-size measure we report \textit{perturbation efficiency}, the mean per-target random-walk influence on the disease module. Decomposing this metric separates direct target--DILI overlap (62\% of Hyperforin's per-target influence) from a modest propagated advantage ($\sim$1.5$\times$), providing a framework for reporting effect size alongside standardized evidence in target-asymmetric network comparisons.

\noindent\textbf{Keywords:} network medicine; proximity Z-score; effect size; statistical evidence; random walk with restart; drug-induced liver injury.
