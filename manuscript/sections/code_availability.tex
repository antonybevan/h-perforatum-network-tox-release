\section*{Code Availability}

All analysis code --- the network-toxicology pipeline, the perturbation-efficiency
metric, the permutation, target--disease overlap (leakage) and Guney-fidelity
analyses, and the figure scripts --- is openly available under the MIT
license at \url{https://github.com/antonybevan/h-perforatum-network-tox}
(manuscript text, figures, and data are released under CC-BY-4.0). The
version used for publication will be archived with a permanent Zenodo DOI from
the tagged release before final submission.
The deposited snapshot includes pinned dependency versions
(\texttt{requirements-lock.txt}; Python 3.12), SHA-256 checksums for every
committed input and result artefact (\texttt{data/CHECKSUMS.sha256}), the full execution graph
(\texttt{scripts/run\_pipeline.py}), and verification scripts for reviewer-evidence,
Guney-fidelity, leakage, and number-consistency checks.
